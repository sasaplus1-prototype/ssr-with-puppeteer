# ssr-with-puppeteer

SSR with [Puppeteer](https://github.com/GoogleChrome/puppeteer)

## Setup

install dependencies.

```console
$ npm install
```

build scripts.

```console
$ npm run build
```

start server(non-ssr mode).

```console
$ npm start
```

start server(ssr mode).

```console
$ SSR=1 npm start
```

open `http://127.0.0.1:3000/` with the browser.
